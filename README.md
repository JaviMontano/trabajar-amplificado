# Trabajo Amplificado · Bootcamp para Managers con Claude Cowork

Material pedagógico abierto de **MetodologIA** para que un manager convierta **Claude Cowork** en un sistema vivo de trabajo: contexto por escrito —reglas, archivos, memoria y cadencias— que viaja contigo.

> **Método + IA = Soberanía.** La IA no reemplaza tu criterio: lo libera para lo que sí importa.
